# Backend Application:

### Repository to the backend: 
https://github.com/jculver1/company_index_backend

### Steps:
git clone git@github.com:jculver1/company_index_backend.git

cd company_index_backend

create table in psql: createdb techstars

knex migrate:latest

knex seed:run

npm install

npm start
